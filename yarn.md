### yarn簡介

(功能和npm一樣，但效能好)

安裝需求

node js

npm

安裝 npm install --global yarn

>  install package

#### npm

npm install 

npm i

#### yarn

yarn

> add package

#### npm

npm install --save react

npm i -S react

#### yarn

yarn add react

> add dev package

#### npm

npm i -D react

#### yarn

yarn add -D react

> remove package

#### npm

npm remove --sace react

#### yarn

yarn remove react

> add global package

#### npm

npm install --global nodepackage

npm i -g nodepackage

#### yarn

yarn global add nodepackage

> run script

#### npm

npm run build

#### yarn

yarn run build
